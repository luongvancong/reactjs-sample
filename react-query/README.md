# Get started

````
$ cp .env.sample .env
$ npm i
$ npm run dev
````

# Multi-language

- Change your content in [this link](https://urlis.net/u7hodrsc).

- Export to csv and replace to `src/lang/language.csv` 

- Run command below

````
$ npm run build:lang
````

# Build production

````
$ npm run build
````
